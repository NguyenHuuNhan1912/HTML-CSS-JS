# CHIỀU RỘNG VÀ CHIỀU CAO TRONG CSS
## Thuộc tính `height` và `width` được sử dụng để đặt chiều cao và chiều rộng của một phần tử dựa trên các giá trị sau:
* Giá trị `length`: Đây là giá trị xác định chiều cao / chiều rộng bằng `đơn vị` trong CSS `(px, em, rem, cm, v.v.)` -> `width/height: 10px;` 
* Giá trị `auto`: Đây là giá trị mặc định `trình duyệt` tự tính toán -> `width/height: auto;`
* Giá trị `%`: Giá trị tương đối xác định chiều cao / chiều rộng theo `phần trăm` của khối chứa phần tử -> `width/height: 20%;`
* Giá trị `inherit`: Đặt chiều cao / chiều rộng `kế thừa` từ phần tử cha -> `width/height: inherit;`
* Giá trị `initial`:  Đặt chiều cao / chiều rộng `thành giá trị mặc định` của nó ->  `width/height: initial;`
